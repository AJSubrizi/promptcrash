# Changelog

All notable changes to PromptCrash will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project uses semantic versioning for published packages.

## [0.1.0] - 2026-05-16

### Added

- Initial PromptCrash monorepo with Next.js dashboard, local SQLite persistence, and TypeScript SDK.
- Redacted crash capture endpoint at `/api/events`.
- Deterministic fallback classification and Vitest regression scaffold generation.
- Demo dataset and dashboard clustering by failure type, route, and prompt version.
- Local-first docs for replay JSON, redaction, and regression scaffolds.
- CI, ESLint, Prettier, Vitest, and package build tooling.

### Package

- Published `@promptcrash/sdk@0.1.0` to npm.
