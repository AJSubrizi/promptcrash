# Code of Conduct

PromptCrash follows the Contributor Covenant Code of Conduct.

## Our Standard

Be respectful, constructive, and focused on improving the project. Harassment, personal attacks, and abusive behavior are not acceptable.

## Enforcement

Project maintainers may remove comments, close issues, or block contributors who violate this code of conduct.

If you experience or observe unacceptable behavior, contact the maintainer through the repository owner profile.
